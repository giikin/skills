# 工作日志 (zerone log)

使用 `zerone log` 命令基于当前仓库的 Git 提交记录生成工作日志，支持日报、周报、月报等多种时间范围。

所有命令需要 `required_permissions: ["all"]`。

## 参数

| 参数 | 说明 | 示例 |
| --- | --- | --- |
| `--since` | 起始时间 | `6.days`、`2024-05-01 00:00:00` |
| `--until` | 截止时间 | `2024-06-01 00:00:00` |

- 支持相对时间（`6.days`）和绝对时间（`2024-05-01 00:00:00`）
- 绝对时间需要用引号包裹
- 不传参数时默认生成今日日报

---

## 日报

```bash
zerone log
```

生成今日的工作日志。

---

## 周报

### 本周（周一至今）

```bash
echo "本周一: $(date -v-$(($(date +%u)-1))d +%Y-%m-%d) ~ 今天: $(date +%Y-%m-%d)"
zerone log --since="$(date -v-$(($(date +%u)-1))d +%Y-%m-%d) 00:00:00"
```

### 上周

```bash
echo "上周一: $(date -v-$(($(date +%u)+6))d +%Y-%m-%d) ~ 上周日: $(date -v-$(($(date +%u)))d +%Y-%m-%d)"
zerone log --since="$(date -v-$(($(date +%u)+6))d +%Y-%m-%d) 00:00:00" --until="$(date -v-$(($(date +%u)))d +%Y-%m-%d) 23:59:59"
```

### 最近 7 天

```bash
echo "$(date -v-6d +%Y-%m-%d) ~ $(date +%Y-%m-%d)"
zerone log --since=6.days
```

---

## 月报

### 本月

```bash
echo "本月: $(date -v1d +%Y-%m-%d) ~ $(date +%Y-%m-%d)"
zerone log --since="$(date -v1d +%Y-%m-%d) 00:00:00"
```

### 指定月份

```bash
# 2024 年 1 月
echo "2024-01-01 ~ 2024-01-31"
zerone log --since="2024-01-01 00:00:00" --until="2024-02-01 00:00:00"
```

---

## 执行步骤

1. **确定时间范围** — 根据用户需求判断日报/周报/月报/自定义范围
2. **打印日期范围** — 先用 `echo` 输出实际的起止日期，确保时间计算正确
3. **执行命令** — 运行对应的 `zerone log` 命令
4. **整理输出** — 根据日志内容整理为结构化的工作报告，标题需包含正确的日期范围

## 报告整理要求

- 按功能模块或项目分组
- 去除重复和无意义的提交（如 merge commit）
- 使用简洁的中文描述每项工作
- 报告标题格式示例：`工作周报（2024-01-08 ~ 2024-01-12）`

## 注意事项

- `date -v` 语法是 macOS 的 BSD date 命令，Linux 下需要使用 `date -d` 替代
- 如果用户提到"这周"且今天是周一，则 `--since` 就是今天
- 确保已正确安装并配置 `@zeronejs/cli`
